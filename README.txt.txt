###phpapplication

Le but de cette application est d'ajouter des utilisateurs en base de donnée et de les affichés.
Elle est faite en PHP, utilisant mySQL pour la partie base de données.
Le fichier docker-compose permet de lancer Apache, MySQL, PhpMyAdmin et PHP, il y a en tout 4 conteneurs.
Il faut importer la base de données depuis l'interface de PhpMyAdmin durant le premier lancement.

###links

Application: http://localhost
PhpMyAdmin: http://localhost:8085
